# Autonomous Email Agent

Phase 1 scaffold for the implementation plan in [`IMPLEMENTATION_PLAN_V2.md`](IMPLEMENTATION_PLAN_V2.md).

## Structure

- `backend/`: FastAPI, SQLAlchemy, Alembic, and the initial database schema
- `frontend/`: Next.js App Router scaffold with Tailwind, Zustand, Axios, and shadcn-style UI primitives

## Next steps

1. Create `backend/.env` from `backend/.env.example`
2. Create `frontend/.env.local` from `frontend/.env.local.example`
3. Install backend and frontend dependencies
4. Run the initial Alembic migration
